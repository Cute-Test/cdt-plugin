h1. Instructions for earlier releases
2 1 






    3
    1
    
    
The CUTE Eclipse plug-in depends on the following software:






    4
    1
    
    






    5
    1
    
    
* "Java5 JRE":http://www.eclipse.org/downloads/moreinfo/jre.php






    6
    1
    
    
* "Eclipse IDE":http://www.eclipse.org/






    7
    1
    
    
* "Eclipse C/C++ Development Tooling plug-in":http://www.eclipse.org/cdt/






    8
    1
    
    
* "Boost C++ Libaries":http://www.boost.org/ or use the Boost Headers provided with the plug-in, with C++0x Boost is no longer needed.






    9
    1
    
    






    10
    1
    
    
For compiling you will also need to have a Eclipse compatible C++ compiler installed:






    11
    1
    
    






    12
    1
    
    
"Official MinGW":http://www.mingw.org/ distribution (currently using GCC 4.x.x)






    13
    1
    
    
"Nuwen MinGW distribution":http://nuwen.net/mingw.html , includes Boost (currently using GCC 4.x.x)






    14
    1

h3. Install the CUTE Eclipse plug-in






    17
    1
    
    






    18
    1
    
    
CUTE depends on the CDT feature Codan. In CDT 7.0 Helios Codan is an optional feature that is not installed by default.






    19
    1
    
    
For p2 to resolve this dependency please add or enable the CDT update site http://download.eclipse.org/tools/cdt/releases/helios/






    20
    1
    
    






    21
    1
    
    
*Install CUTE using the Eclipse Marketplace*






    22
    1
    
    
This is the easiest way to get Cute, choose _Help > Eclipse Marketplace_  in the Eclipse menu and search for Cute. But you will miss out on the optional features of Cute and 






    23
    1
    
    
the Refactoring Previews. But you can install them later using the CUTE update site.






    24
    1
    
    






    25
    1
    
    
*Install CUTE using the CUTE update site.*






    26
    1
    
    
!>{width:466px}/attachments/download/2/cuteInstall.png!:/projects/cute/wiki/cuteInstall.png






    27
    1
    
    
# Choose _Help > Install New Software_ in the Eclipse menu to start the Eclipse software installer






    28
    1
    
    
# Click _Add.._ to add the CUTE repository






    29
    1
    
    
# Specify the CUTE update site, http://www.cute-test.com/updatesite. The site is added to the list of Available Software.






    30
    1
    
    
# Choose at least the Cute Plugin and one of the Cute Headers features.






    31
    1
    
    
# Click the Next button. Eclipse will then resolve dependencies and show the install dialog.






    32
    1
    
    
# Read and accept the licences and only click Finish if you want to continue installing Cute and agree with the licences.






    33
    1
    
    






    34
    1
    
    
Eclipse will now download and install everything you need to use CUTE in Eclipse (this may take a while..). Eclipse will need to restart after the setup. If you want to modify the CUTE Eclipse plug-in, you will need the source code.
