CUTE Standalone¶

The standalone version includes a test suite and an example with deliberately failing tests. If you just want to use CUTE, all you need is the header files in cute/cute_lib.

Eclipse users may be interested in the standalone version for the examples and test suite.

Downloads are provided in zip and tar versions. The content of the two versions is identical.
