Instructions for earlier releases¶

Helios / CDT 7.0¶

The CUTE Eclipse plug-in depends on the following software:

  • Java5 JRE
  • Eclipse IDE
  • Eclipse C/C++ Development Tooling plug-in
  • Boost C++ Libaries or use the Boost Headers provided with the plug-in, with C++0x Boost is no longer needed.

For compiling you will also need to have an Eclipse compatible C++ compiler installed:

Official MinGW distribution (currently using GCC 4.x.x)
Nuwen MinGW distribution , includes Boost (currently using GCC 4.x.x)
Note that there are many Eclipse compatible compilers availible that are not based on the MinGW distribution.

Install the CUTE Eclipse plug-in¶

CUTE depends on the CDT feature Codan. In CDT 7.0 Helios Codan is an optional feature that is not installed by default.
For p2 to resolve this dependency please add or enable the CDT update site http://download.eclipse.org/tools/cdt/releases/helios/

Install CUTE using the CUTE update site.
  1. Choose Help > Install New Software in the Eclipse menu to start the Eclipse software installer
  2. Click Add... to add the CUTE repository
  3. Specify the CUTE update site, http://www.cute-test.com/updatesite. The site is added to the list of software available.
  4. Uncheck "Show only the latest versions of available software".
  5. Choose at least the Cute Plugin version 4.1.2 and one of the Cute Headers features.
  6. Click the Next button. Eclipse will then resolve dependencies and show the install dialog.
  7. Read and accept the licences and only click Finish if you want to continue installing Cute and agree with the licences.

Eclipse will now download and install everything you need to use CUTE in Eclipse (this may take a while). Eclipse will need to be restarted after the setup.
