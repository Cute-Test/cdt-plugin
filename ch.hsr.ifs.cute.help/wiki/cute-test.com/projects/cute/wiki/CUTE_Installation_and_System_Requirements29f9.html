CUTE Installation and System Requirements¶

You can use the CUTE unit testing framework with the Eclipse IDE or standalone. If you use Eclipse for software development, the CUTE Eclipse plug has everything you need. If you use Microsoft Visual Studio, install the CUTE standalone version.

CUTE Standalone or with Microsoft Visual Studio¶

Download the CUTE standalone source files and add them to your source tree.

CUTE works with modern GNU and Microsoft C++ tool chains, plus the Boost C++ Libraries:

Tool Chain Version Notes
GNU g++ 4.1.1, binutils-2.16, Boost 1.35
Microsoft Visual Studio TODO, Boost 1.35

CUTE Eclipse Plug-in¶

The latest release of CUTE is version 4.2.2 for Eclipse Indigo and CDT 8.0 - If you are working with Eclipse Helios and CDT 7.0 checkout the instructions for earlier releases.

The CUTE Eclipse plug-in depends on the following software:

  • Java5 JRE
  • Eclipse IDE
  • Eclipse C/C++ Development Tooling plug-in
  • Boost C++ Libaries or use the Boost Headers provided with the plug-in, with C++0x Boost is no longer needed.

For compiling you will also need to have a Eclipse compatible C++ compiler installed:

Official MinGW distribution (currently using GCC 4.x.x)
Nuwen MinGW distribution , includes Boost (currently using GCC 4.x.x)
Note that there are many Eclipse compatible compilers availible that are not based on the MinGW distribution.

Install the CUTE Eclipse plug-in¶

The CUTE plug-in can either be installed from the Eclipse Marketplace or by using our update-site directly.

Installing CUTE using the Eclipse Marketplace
This is the easiest way to get Cute, choose Help > Eclipse Marketplace in the Eclipse menu and search for Cute. But you will miss out on the optional features of Cute and
the Refactoring Previews. But you can install them later using the CUTE update site.

Installing CUTE using the CUTE update site.
  1. Choose Help > Install New Software in the Eclipse menu to start the Eclipse software installer
  2. Click Add.. to add the CUTE repository
  3. Specify the CUTE update site, http://www.cute-test.com/updatesite. The site is added to the list of Available Software.
  4. Choose at least the Cute Plugin and one of the Cute Headers features.
  5. Click the Next button. Eclipse will then resolve dependencies and show the install dialog.
  6. Read and accept the licences and only click Finish if you want to continue installing Cute and agree with the licences.

Eclipse will download and install everything you need to use CUTE in Eclipse (this may take a while.). Eclipse will need to restart after the setup.
